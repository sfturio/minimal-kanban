1. Overview

Minimal Kanban (Fluxo Essencial) e uma aplicacao Kanban minimalista focada em produtividade pessoal e organizacao de tarefas, com arquitetura modular no frontend e sincronizacao cloud via Supabase.

O objetivo do projeto e oferecer um Kanban rapido, simples e extensivel, evitando complexidade desnecessaria comum em ferramentas tradicionais.

Principais caracteristicas

  • Interface minimalista e rapida
  • Arquitetura modular escalavel
  • Persistencia hibrida (local + cloud)
  • Automacao de criacao de tarefas
  • Base preparada para evolucao SaaS

2. Architecture Overview

A aplicacao utiliza uma arquitetura frontend modular baseada em ES Modules, organizada por responsabilidade.

src/
 +-- app.js
 +-- state/
 +-- services/
 +-- ui/
 +-- features/
 +-- storage/
 +-- utils/

Entry Point

src/app.js

  • Inicializacao da aplicacao
  • Bootstrap do estado global
  • Inicializacao da autenticacao
  • Carregamento inicial de boards/tasks

3. Technology Stack

Frontend

  • HTML
  • CSS
  • Vanilla JavaScript (ES Modules)

Motivacao: simplicidade, controle total do runtime e ausencia de dependencias pesadas.

Persistencia

  • Supabase
  • PostgreSQL
  • Auth
  • Row Level Security (RLS)
  • Local Storage

Local Storage e utilizado como cache local, fallback offline e compatibilidade com estado legado.

8. Database Architecture (Supabase)

Modelagem escalavel preparada para colaboracao futura.

Tabelas

  • profiles
  • boards
  • columns
  • tasks
  • task_tags
  • comments

Relacoes

  • profiles.id -> auth.users.id
  • boards.user_id -> profiles.id
  • columns.board_id -> boards.id
  • tasks.board_id -> boards.id
  • tasks.column_id -> columns.id
  • task_tags.task_id -> tasks.id
  • comments.task_id -> tasks.id
  • comments.user_id -> profiles.id

Indices principais

  • tasks(board_id)
  • tasks(column_id)
  • columns(board_id)

10. Sync Architecture

Sincronizacao hibrida entre aplicacao e banco:

  1. Usuario autentica
  2. Snapshot cloud e carregado
  3. Estado local e reconciliado
  4. Alteracoes sao enviadas com debounce
