# Fluxo Essencial

Um Kanban pessoal, simples e rápido.

A ideia é ser um espaço de foco: sem excesso de recursos, sem distração e com tudo funcionando direto no navegador.

## O que ele faz

- Tabelas (criar, renomear, excluir)
- 3 colunas fixas: `Próximos`, `Em andamento`, `Concluído`
- Criar tarefas com título e categoria opcional
- Arrastar e soltar com indicador visual de posição
- Renomear e excluir tarefas sem popup do navegador
- Botão `Planejar` para gerar várias tarefas de uma vez (`;`)
- Se usar parênteses, vira categoria (ex.: `Estudar JS (noite)`)
- Modo foco
- Tema escuro/claro (escuro por padrão)
- Persistência local com `localStorage`

## Como usar

1. Abra o `index.html` (ou rode com Live Server).
2. Crie uma tarefa normalmente ou use `Planejar`.
3. Separe múltiplas tarefas com `;`.

Exemplo:

```text
revisar e-mails ; treino de peito (academia) ; estudar JavaScript (noite)
```
